# TrackFlow-SA

Nomes dos membros:Samuel Elias Candinho, Roberto Stockhausen Hille, Lucas Viera Pedroso , Daniel Tonioti e Natan Vitor Dré Avanço Barbosa

Fazer um aplicativo com o objetivo simular um sistema interno de uma empresa ferroviaria para gerar relatorios sobre os estados dos seus trens e administradores

Funções Prevista:
logar(Conseguir colocar email e senha para entrar no aplicativo)
Adicionar sensosres(conseguir adicionar um novo sensor)
Remover sensores(Conseguir remover um sensor)
