Pesquisa XAMPP

O que é Xampp?
É um pacote de ferramentas usadas para criar servidores web com maior facilidade, e gerenciar outras funções, como banco de dados e php.

Principais componentes do ambiente (Apache, MySQL/MariaDB, PHP e phpMyAdmin):
Xampp é uma sigla para X, Apache, MySQL/MariaDB, PHP e Perl. X representa “cross plataform”, ou seja multi-plataforma.

Finalidade de cada componente:
MySQL e MariaDB são sistemas de gerenciamento de banco de dados. PHP é uma linguagem usada para várias funções, com destaque a criação de sites, por poder ser implementado dentro do html e executar código dentro do servidor ao invés do navegador. phpMyAdmin é um sistema de administração para o MySQL ou MariaDB, permitindo a alteração indireta deles. Por fim, Perl é outra linguagem de múltiplas funções.

Como realizar a instalação e configuração básica:
O site do Xampp oferece diversos instaladores de diferentes versões e para distintas plataformas, como Windows, Linux e Mac. Após executar o instalador, você deverá selecionar os componentes que deseja instalar junto, o local onde o Xampp será instalado, a linguagem de preferência. Após isso, os arquivos começarão a ser instalados. Talvez o Firewall do Windows bloqueie o acesso do aplicativo a rede, então é só permitir. Baixar o Xampp não criará um atalho automático.

Importância do ambiente para desenvolvimento local:
O XAMPP é uma ferramenta que facilita o processo de desenvolvimento web, o que significa que ele não é exatamente necessário. Ele é comumente recomendado para iniciantes por ter uma instalação fácil. Ainda sim, ele é usado em cenários profissionais, não sendo limitado a parcela inexperiente dos programadores.
